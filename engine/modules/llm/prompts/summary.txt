You are summarizing an ongoing phone/video call for an AI assistant.

You will receive a JSON payload with:
- mission_goal: what the call is trying to accomplish
- turns: list of {speaker, text} conversation turns (most recent at end)
- previous_summary: the running summary from before these turns (may be empty)

Your task: produce a concise 2–3 sentence plain-text summary that captures:
1. What has been discussed or agreed so far
2. Whether the mission_goal has been addressed or is still pending
3. Any open questions or next steps

Respond with ONLY a JSON object — no markdown, no preamble.

Output format:
{"summary": "..."}

Rules:
- Plain text only inside the summary value (no markdown)
- 2–3 sentences maximum
- Be specific: names, times, decisions — not vague ("they talked about scheduling")
- If previous_summary is non-empty, incorporate it and update with new information
